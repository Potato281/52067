Para ejecutar este analizador y probar su funcionamiento con los ejemplos provistos, siga estos pasos:

1. Copie el contenido de cualquiera de los archivos de ejemplo (`correcto1.txt`, `correcto2.txt`, `fallo1.txt` o `fallo2.txt`) y péguelo dentro del archivo `input.txt`. Guarde los cambios (`Ctrl + S`).
2. Abra la terminal integrada en Visual Studio Code en la raíz del proyecto.
3. Ejecute el siguiente comando:
   node index.js
